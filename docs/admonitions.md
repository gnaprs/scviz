Admonitions that use the inline modifiers must be declared prior to the content block you want to place them beside

note, abstract, info, tip, success, question, warning, failure, danger, bug, example, quote

Link here: <https://squidfunk.github.io/mkdocs-material/reference/admonitions/#inline-blocks-inline>
